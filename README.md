# 💼 HR Employee Attrition Prediction

A Machine Learning web app that predicts whether an employee is likely to leave the company based on key features.

---

## 🚀 Live Demo
👉 https://your-app-name.streamlit.app

---

## 📌 Project Overview

Employee attrition leads to increased hiring costs and productivity loss.  
This project uses Machine Learning to predict employee attrition (Yes/No) and help organizations take proactive decisions.

---

## 🧠 Features Used

- Age  
- Monthly Income  
- Job Level  
- Job Satisfaction  
- Work-Life Balance  
- Years at Company  
- OverTime  

---

## 🤖 Model

- Algorithm: Random Forest Classifier  
- Preprocessing:
  - Label Encoding  
  - Standard Scaling  
- Accuracy: ~80-85%

---

## 🛠️ Tech Stack
